Project

Custom Pixel ROM Installation on Samsung S10 5G

This project involved installing a custom Pixel Experience ROM on a Samsung Galaxy S10 5G. The goal was to replace the stock Samsung One UI with a clean, near-stock Android Pixel experience, known for its streamlined interface and timely updates.

Key Aspects:

  • Device: Samsung Galaxy S10 5G
  • ROM: Pixel Experience
  • Objective: Achieve a clean Android experience, improve performance , and potentially gain access to faster updates.

My Contributions:

  • Researched and selected a stable and reputable Pixel Experience ROM for the S10 5G.
  • Successfully unlocked the bootloader of the Samsung S10 5G.
  • Installed a custom recovery (TWRP) using appropriate tools and procedures.
  • Flashed the Pixel Experience ROM and any necessary supplementary packages
  • Configured the system and tested its functionality, including core features, camera, and network connectivity
  • Troubleshooting any issues encountered during the installation process.

Outcome:

Successfully replaced the stock ROM with Pixel Experience, resulting in an increased bench score(smoother interface) and Google's Circle-to-Search Feature. Although VoLTE(Voice-over-LTE) was disabled, It didn't remove much from the functionality of Voice calls. This project demonstrated my ability to navigate the complexities of Android system modification and my understanding of custom ROM installation processes.

Project

Hackintosh High Sierra Installation on HP Pavilion dv6

This project involved installing macOS High Sierra on a non-Apple computer (HP Pavilion dv6). This is known as a "Hackintosh" and requires careful hardware selection, BIOS configuration, and customization.

Key Aspects:

  • Hardware: HP Pavilion dv6
  • CPU: Intel Core i5 2410M @ 2.30GHz
  • RAM: 6GB
  • Storage: 256GB SSD
  • macOS: macOS High Sierra (10.13)
  • Bootloader: Clover

My Contributions:

  • Researched hardware compatibility to ensure a successful installation.
  • Configured the laptop's BIOS settings for macOS compatibility (e.g., AHCI mode, disabling Secure Boot).
  • Created a bootable USB installer using the Hackintosh Zone distribution.
  • Manually selected and installed necessary kexts (kernel extensions) for hardware drivers (e.g., audio, Wi-Fi, Ethernet, graphics).
  • Configured the bootloader for proper startup and system stability.
  • Troubleshooting and resolving installation errors and post-installation issues.
  • Optimized system performance and stability.

Outcome:

Successfully installed and configured macOS High Sierra on the HP Pavilion dv6, resulting in a functional Hackintosh system. This project demonstrates my ability to work with complex system configurations, troubleshoot hardware and software incompatibilities, and achieve a working macOS environment on unsupported hardware.

Project

iMac Processor Upgrade and macOS Installation

This project involved upgrading the processor in a 2007 iMac and installing a newer version of macOS than officially supported.

Key Aspects:

  • iMac Model: 2007 iMac 7,1
  • Original CPU: Intel Core 2 Duo Penryn 2.7ghz
  • Upgraded CPU: Newer Intel Core 2 Duo Penryn 2.8ghz
  • macOS: MacOS Catalina
  • Patching Tool: dosdude1's macOS Patching Tool

My Contributions:

  • Disassembled the iMac to access and replace the CPU.
  • Installed the more capable Penryn processor.
  • Used dosdude1's patcher tool to create a modified macOS installer.
  • Installed the patched macOS on the upgraded iMac.
  • Troubleshooting any compatibility issues or errors during the installation.
  • Testing and verifying the functionality of the upgraded system.

Outcome:

Successfully upgraded the iMac's processor and installed a newer version of macOS, extending the life of the machine and making it compatible with newer softwares . This project demonstrates my hardware modification skills and my ability to utilize community tools to overcome manufacturer limitations.
